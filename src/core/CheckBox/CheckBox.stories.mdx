# CheckBox

Displays a CheckBox.

export const Template = args => (
    <div>
        <CheckBox {...args} />
    </div>
);

<Preview>
    <Story name="CheckBox" args={{ size: 'S' }}>
        {Template.bind({})}
    </Story>

### Color Variations

You can either change the theme or pass the colors as props.

<Preview withToolbar>
    <div>
        <CheckBox checked={true} bgColor="#37d5d3" />
        <CheckBox checked={true} bgColor="#2a0481" />
        <CheckBox checked={true} bgColor="#37d5d3" labelColor="blue" />
        <CheckBox checked={true} bgColor="#2a0481" labelColor="red" />
    </div>
</Preview>

### Error

Displays Error Text if hasError={true}.

<Preview withToolbar>
    <div>
        <CheckBox hasError={true} errorText="Please check the CheckBox" />
    </div>
</Preview>
